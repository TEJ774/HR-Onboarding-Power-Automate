## Overview
This repository demonstrates a real-world HR onboarding automation built using Microsoft Power Automate.
The automation focuses on improving efficiency, introducing structured approvals, and providing visibility and auditability into the onboarding process.

## Business Problem
HR onboarding was previously handled through emails and manual coordination between HR, managers, and IT. This caused delays, inconsistent approvals, and limited tracking.

## Solution
A Power Automate flow was created to:
- Capture onboarding requests
- Route approvals to managers
- Notify stakeholders
- Provide a clear audit trail

## Tools Used
- Microsoft Power Automate
- Microsoft Forms or SharePoint
- Outlook / Teams
